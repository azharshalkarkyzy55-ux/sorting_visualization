#Сұрыптау алгоритмдері (Java)

Сұрыптау алгоритмдері — мәліметтерді (массив, тізім) белгілі бір тәртіппен (өсу немесе кему) орналастыру тәсілдері. Олар іздеуді жылдамдату, деректерді реттеу үшін дерекқорларда, графикада және жасанды интеллектте қолданылады. Негізгі түрлері: көпіршікті, енгізу, таңдау, жылдам (Quick sort) және біріктіру (Merge sort).
Бұл жоба Java тілінде екі қарапайым сұрыптау алгоритмін көрсетуге арналған.

Қолданылған алгоритмдер:
- Bubble Sort (көпіршік арқылы сұрыптау)
- Selection Sort (таңдау арқылы сұрыптау)

Bubble Sort қалай жұмыс істейді:
Бұл алгоритм көршілес сандарды бір-бірімен салыстырады. Егер сол жақтағы сан үлкен болса, олар орын ауыстырады. Осы процесс бірнеше рет қайталанып, ең үлкен сандар біртіндеп соңына жылжиды.

Selection Sort қалай жұмыс істейді:
Бұл алгоритм әр қадамда массив ішінен ең кіші санды табады. Содан кейін оны бірінші бос орынға қояды. Кейін келесі ең кіші санды тауып, солай жалғасады.

Нәтиже:
Бағдарлама массивтің қалай өзгеретінін қадам-қадамымен консольда көрсетеді. Осы арқылы сұрыптау процесі жақсы түсінікті болады.

Қорытынды:
Бұл екі алгоритм де қарапайым, бірақ үлкен массивтер үшін онша тиімді емес. Екеуінің де күрделілігі O(n^2)
